HTML5

HTML é uma abreviação para Hypertext Markup Language. Uma Markup Language ou uma linguagem de marcação, não é uma linguagem de programação. É um tipo de linguagem que usa tags especiais para anotação e marcação de documentos. A primeira versão do html foi criada em 1993, e com o passar dos anos veio sofrendo atualizações, sua versão atual é o html 5.

Todo documento html deve começar com a tag <!DOCTYPE html >, para que o browser saiba que aquele é um documento html 5. Logo em seguida você deve usar a tag <head> </head>, dentro dela você pode usar as tags:
<title> </title> Para determinar o título da pagina.
<meta> para determinar a codificação dos caracteres da página.
<link> para linkar style sheets.

Feito isso você dever usar a tag <body> </body>. Dentro dessa tag vai o conteúdo da sua página, e você deve fazer uso de todas as tags disponíveis para estilizar a sua pagina, e incluir o conteúdo que deseja. Abaixo estão exemplificados algumas das tags que podem ser utilizadas em um documento html:

As tags de cabeçalho (h1 a h6) são utilizadas para destacar um texto, muitas vezes usadas como titulo de uma seção.

<h1> </h1>

Esse é um header h1

<h2> </h2>

Esse é um header h2

<h3> </h3>

Esse é um header h3

<h4> </h4>

Esse é um header h4

<h5> </h5>
Esse é um header h5
<h6> </h6>
Esse é um header h6
Você deve usar as tags <p></p> para delimitar parágrafos. Além de texto, você também pode inserir outros elementos dentro dessa tag como imagens, links e etc.
As tags b, u e i são usadas para deixar texto em negrito, italico e colocar underline respectivamente.
<b></b> Texto em negrito.
<i></i> Texto em italico.
<u></u> Texto com underline.
As tags img e a são usadas para colocar imagens e links no seu documento html. Na tag img você deve usar o atributo src para inserir o endereço da imagem, e as tag alt e title para inserir um texto alternativo que aparece ao colocarmos o cursor em cima da imagem. Na tag a você deve usar o atributo href para inserir o endereço do link, e pode usar o atributo target com o valor "_blank" para que o link seja aberto em uma nova aba.
<img>
<a></a> Link para a página incial do google.

Algumas tags servem como containers que encapsulam varios elementos html dentro dela. Algumas dessas tags são <header></header> <nav></nav> <section></section> <article></article> <aside></aside> <footer></footer>. Uma dica importante é agrupar elementos dentro da tag correspondente a sua posição dentro da página, seguindo o mapeamento exemplificado na figura. Além disso, um container mais genérico muito utilizado é o <div></div>.

Quando um browser carrega um documento html, ele cria um DOM da página. DOM é uma abreviação do termo Document Object Model, que é uma representação do documento html em uma estrutura de arvore. Onde cada um dos nós dessa arvore representa uma tag utilizada no documento. É através dessa estrutura que o javascript consegue manipular um documento html.

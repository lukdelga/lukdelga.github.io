CSS3

A abreviação CSS significa Cascade Style Sheet, e é a linguagem utilizada para estilizar um documento html. A linguagem CSS descreve como elementos html devem ser exibidos.

Existem 3 formas de utilização da linguagem css em um documento html. O primeiro, é inserir o codigo na seção <head> </head> do documento html. Você deve incluir a tag <style> </style>, e todo o código css vai dentro dela.
ex: 


                    <!DOCTYPE html>
                    <head>
                        <meta charset="UTF-8">
                        <title>...</title>
                        <style>
                        codigo css aqui!
                        .
                        .
                        .
                        </style>
                    </head>
Outra opção, é colocar o css dentro da tag, através do atributo style.
ex:

                    <h1 style="font-weight: normal;">Título do texto</h1>
A maneira mais recomendada é colocar todo o codigo em um arquivo .css separado. Para isso é necessário linkar o arquivo .css com o arquivo .html. Isso é feito pela tag link na cabeça do documento html.
ex:
documento .html 
                    <!DOCTYPE html>
                    <head>
                        <meta charset="UTF-8">
                        <title>...</title>
                        <link rel="stylesheet" href="mystylesheet.css">
                    </head>
documento .css 
                        seletor1 {
                            atributo1: valor;
                            atributo2: valor;
                            .
                            .
                            atributoN: valor;
                        }
                        .
                        .
                        seletorN {
                            atributo1: valor;
                            atributo2: valor;
                            .
                            .
                            atributoN: valor;
                        }
Para estilizar os diferentes elementos de um documento html através de um arquivo css, é necessário usar um seletor que defina qual elemento da pagina você deseja estilizar, seguido de {}. Dentro das chaves você deve colocar os atributos que deseja modificar e seus respectivos valores separados por : e usando ; para determinar o fim da declaração (descrito no exemplo anterior). O seletor pode ser definido pela tag do elemento que se deseja modificar. O problema com essa solução é que todos os elementos daquele tipo serão alterados, e muitas vezes precisamos estilizar instancias diferentes de um mesmo elemento de formas diferentes. Pra isso, a linguagem css disponibiliza as classes e os ids, que podem ser atribuídos aos elementos do documento html, através dos atributos class e id respectivamente.
ex:
documento .html 
                        .
                        .
                        .
                        <body>
                            .
                            .
                            <h1 class="titulo-maior">Título do texto</h1>
                            <p id="texto-principal"> Lorem ipsum dolor sit amet consectetur adipisicing elit. Velit, vitae! </p>
                            .
                            .
                        </body>
Documento .css 
                        .titulo-maior {
                            font-size: 8em;
                            font-weight: bolder;
                        }

                        #texto-principal {
                            text-align: justify;
                        }

Uma parte muito importante da linguagem CSS é o Box Model. Todos elementos se comportam como uma caixa que tem padding (espaço entre o conteúdo e a borda), border (borda do elemento), margin (o espaço entre a borda de um elemento e seus elementos vizinhos). Cada um dos lados dos atributos da caixa podem ser estilizados individualmente.
